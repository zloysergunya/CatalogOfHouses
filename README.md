# Мобильный каталог домов
Мобильное приложение "Мобильный каталог домов (МКД)" отображает сводную информацию о жилом фонде управляющих компаний, входящих в холдинг, на текущую дату (по умолчанию) на основании данных из Мастер справочника.

Для получения сводной информации по различным показателям можно воспользоваться фильтрами.

Для получения детальной информации по отдельному дому предусмотрена возможность поиска.
